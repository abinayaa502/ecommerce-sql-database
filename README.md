# E-Commerce SQL Database Project

This repository contains an intermediate-level SQL project featuring a complete
e-commerce database design with schema creation, sample data, queries,
stored procedures, triggers, and reporting views.

---

## Database Features

### ✔ Database Schema
- customers
- products
- orders
- order_items
- payments

Includes:
- primary keys
- foreign keys
- NOT NULL constraints
- UNIQUE constraints
- CASCADE DELETE

### ✔ Sample Data
Pre-populated tables for realistic query testing.

---

## SQL Concepts Demonstrated

### Queries
- JOINs (INNER, LEFT, RIGHT)
- Subqueries
- Aggregate Functions
- Window Functions

### Advanced Features
- Stored Procedures
- BEFORE UPDATE Trigger (Audit logging)
- Reporting Views
